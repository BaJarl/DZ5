Селектори бувають

  1. Універсальний селектор (*) - застосовує стилі до усіх елементів сторінки
  2. За елементом - (p, h1, ol і тд.) - застосовує стилі для усіх вказаних елементів
  3. По ідентифікатору - прописуємо ID для елемента ( p id="ID1"), потім в css через #ID1 {властивості}. Елемент з вказаним ідентифікатором буде мати прописані властивості
  4. За класом - прописуємо клас для елемента ( p class="class1"), потім в css через .class1 {властивості} клас можна назначати для багатьох елементів на відміну від ID - унікального ідентифікатору
  5. За кількома класами, елементами - вказуємо через зап'яту перелік для застосування для них стилів (h1, .class1, .class2, p)
  6. Вкладений селектор - застосовує для нащадків стилі (.class1 > p). В даному випадку для усіх (р), які є нащадками класу .class1
  7. Вибірковий селектор - (h1#ID1) стилі будуть застосовані для h1 з ідентифыкатором ID1
  8. Сусідній селектор - застосовує стилі для h1 .class1, які біля h1 #ID1 (h1#ID1 + h1.class1)
